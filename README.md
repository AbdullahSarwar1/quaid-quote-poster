# Quaid-e-Azam Tribute Poster

A clean, high-performance, and visually balanced single-page web poster dedicated to the founder of Pakistan, Muhammad Ali Jinnah. This project elegantly showcases a famous historical quote from 1937 utilizing refined typography, subtle container drop-shadows, and an optimized fluid layout.

## Features & Optimizations
* **Fluid Layout Mechanics:** Relies on natural text wrapping and fluid dimensions instead of hard formatting constraints, allowing for a highly stable response across varying container widths.
* **Asset Performance Optimization:** Image assets (including the watercolor background) are compressed and converted to lightweight formats to ensure sub-second rendering speeds on both desktop and mobile devices.
* **Visual Depth Styling:** Leverages deep CSS `box-shadow` structures and border radiuses on image components to create a framed, gallery-poster aesthetic that pops against gradients.
